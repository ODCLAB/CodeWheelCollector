```cpp
key_t ftok(const char *pathname,int proj_id)
```
- 作用
  把一个已存在的路径名和一个整数标识符转换成IPC键值。通常用于`消息队列`、`共享内存`

- 参数

  | 名称     | 意义           |
  | -------- | -------------- |
  | pathname | 已存在的路径名 |
  | proj_id  | 计划代号       |

- 返回值

  | 值      | 意义                        |
  | ------- | --------------------------- |
  | %key_t% | IPC键值                     |
  | -1      | 出错。错误原因存于`error`中 |
